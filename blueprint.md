# 自分用デイリーニュースサイト

## 概要

日々の情報収集のために

- 今日出版された論文 (5 件程度)
- 今日更新されたニュース (10 件程度)

を確認できるサイトを作成する。

イメージとして、 Google の Discover をリッチにしたもの。

## 構成

- GitHub Actions cron: コアの実行
- Vercel: デプロイ
- OpenAI API Key: 論文要約用

## 詳細

- GitHub Actions cron で毎朝定期実行
- いくつかのニュースサイトを巡回し、興味のありそうなページをスコアリングで選定
  - 自作キーボード、 rust, python, claude といったキーワードに反応
  - 可能であればチェックしたサイトの情報を元にスコアリングを強化し、より自分用としての機能を向上したい
- Arxiv で興味のありそうな論文を選定
  - combinatorial optimization, NP-hard, heuristic などを abstract に含んでいるか確認、スコアリング
  - OpenAI API Key を使って論文を日本語で要約する
- 各情報は JSON 形式でレポジトリに保管
  - 日付
  - タグ
  - 概要
  - リンク
  - スコア
  - タイトル
  - 形式 (paper, news, ...)
  - その他必要な情報
- 保管された情報を Next.js などで今日分のサイト一覧をウェブページで閲覧できるようにし、 vercel でデプロイ
  - 要件
    - 今日分の一覧の表示
    - タグ別の一覧の表示
    - 簡単な検索機能
